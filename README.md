# MacGame

An insanely fun mini-runner starring Mac. Jump, punch, and collect orbs to crank your combo.

## Play locally
Open `index.html` in a browser.

## Mobile + PWA
- The on-screen buttons appear on mobile automatically.
- Add to Home Screen after visiting the site once to install the offline PWA.

## Deploy to GitHub Pages
1. Create a new GitHub repo and add these files.
2. In the repo settings, enable **Pages**.
3. Select the branch (usually `main`) and root folder.
4. Save and use the provided URL.

Sprites live in `sprites/`.
